# Store Management System

A Store Management System built with Next.js and MongoDB.

## Tech Stack

- Next.js 16
- React
- MongoDB Atlas
- Mongoose
- Tailwind CSS

## Features

### Product Module

- Create Product
- View All Products
- View Single Product
- Update Product
- Delete Product

## API Endpoints

GET    /api/products
POST   /api/products
GET    /api/products/:id
PUT    /api/products/:id
DELETE /api/products/:id
